# FishAudio 历史探测

## 结论状态

Fish Audio 做过较多标签和双人对话探测，但当前不作为主线：

- Fish 原生多说话人能力受模型/额度限制；没有 `s2-pro` 时，正式双人对话不可依赖。
- 旧的 Fish 情绪标签经验仍有参考价值，但 MiniMax 主线不直接使用 Fish 方括号标签。
- 父亲音色 clone 台词可作为未来重新录制参考。

## 使用规则

如果重新启用 Fish，先做 6-8 turn 短测，不直接套旧参数生成整期。
