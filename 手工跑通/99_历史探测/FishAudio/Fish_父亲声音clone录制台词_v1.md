# Fish 父亲声音 clone 录制台词 v1

> 用途：给 Fish Audio 重新 clone / persistent voice 父亲音色。目标不是录成“情绪表演样片”，而是录出稳定、温和、有思考感的父亲声音，让后续脚本里的 Fish S2 情绪标签负责具体表演。

## 录制原则

- 只录一个人，环境安静，无背景音乐、电视、人声、空调风扇声。
- 手机录音可以，麦克风离嘴大约一掌距离，放稳，不要手持晃动。
- 每段 15-25 秒，建议录 3 段；如果只想先快速试，至少录第 1 段。
- 每句话之间自然停半秒，不要赶。
- 语气要有温度，但不要大喊、夸张惊讶、忽高忽低。
- 上传时尽量附上对应文本，标点要和实际朗读一致。

## 第 1 段：主音色，温和自然

来，咱们慢慢聊一个有点奇怪的事。我第一次看到它的时候，也没有马上想明白。它不是那种一听就有答案的问题，反而像一根小刺，卡在脑子里。你可以先不用急着判断对不对，咱俩把它拆开，一点一点看。

## 第 2 段：压低声音，反常识钩子

我跟你说，最怪的地方在这儿。远远看过去，那是一大片树林，有很多很多根树干。可是科学家说，它们可能不是很多棵树，而是一棵树在地底下分出来的样子。嗯……这句话听起来就有点绕，对吧？

## 第 3 段：轻微担心，但克制

后来我读到后面，心里稍微沉了一下。因为老的树干还在，可新的小树干不太长得起来。这样一想，它好像不是突然坏掉，而是慢慢接不上了。我不想把它说得太可怕，但这个问题确实值得停一下。

## 上传/命名建议

- 标题：`爸爸_温和探索_v1`
- Visibility：`private`
- 如果支持多段样本：三段都上传，并分别填入精确文本。
- 如果只能先传一段：优先传第 1 段；父亲音色稳定后，再补第 2、3 段扩展范围。

## 后续测试句

clone 完后，用下面这段做 20-30 秒测试，不要直接上长音频：

```text
<|speaker:0|>[温和，像自己也在想] 我第一次看到这句话，也没马上转过来。
<|speaker:1|>[惊讶，短促反问] 啊？一片树林，是一棵树？
<|speaker:0|>[压低声音，像讲一个反常识发现] 对，怪就怪在这儿。它地上看起来是很多根，地下可能是一整套连着的根。
<|speaker:1|>[试探着猜，尾音上扬] 所以……像一只手伸出来好多根手指？
<|speaker:0|>[轻轻笑一下，像被这个比方点通了] 对，你这个比方还挺贴。手指分开，可下面还是一只手。
```
