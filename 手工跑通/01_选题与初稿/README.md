# 01 选题与初稿

## 目标

把当天候选新闻跑成一篇符合「喂，今天」小学高年级档的正文初稿。这个阶段不处理 TTS 标签、音频停顿和声音参数。

## 输入

- `PRD_v0.6.md`
- `Content_Safety_Policy_v0.1.md`
- `prompts/小学高年级_写稿提示词_当前最高版本.md`（当前运行版：`v0.10`）
- 当天新闻材料和家长选择结果

## 步骤

1. 先给 3 个候选，标 1 个推荐，家长拍板。
   - 候选新闻源不能只从单一官方科技/成果报道里取；每天至少混合 2 类来源口径，例如生活消费、自然/动物、国际科普、文博、体育、城市生活、青少年可感知的社会新闻。
   - 官方通稿、科技日报、部门成果稿可以作为事实源之一，但不应成为默认选题池；若使用这类源，必须说明儿童听感入口是什么。
   - 每个候选用一句话标明“孩子为什么会想听”，不能只写成人视角的意义。
2. 按当前 `prompts/` 版本写整稿。
3. 做事实、风险、产品宪法自检。
4. 只改正文质量，不提前写 TTS 控制词。

## 产物

- `稿子/日期_话题_小学档.md`
- 稿内保留元数据、选题记录、自检、风险提示。

## 通过标准

- 一期只讲一条主题新闻。
- 3 个候选之间有来源和题材多样性，不是同一类官方成果稿换标题。
- 爸爸问句多于结论。
- 没有教育总结、标准答案、北极星显性词。
- 家长能接受这个话题和讲法。

## 沉淀位置

稳定经验写回 `prompts/`；产品级决策才写回 `PRD_v0.6.md` 或 `Content_Safety_Policy_v0.1.md`。
