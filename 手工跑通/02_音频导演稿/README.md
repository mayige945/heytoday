# 02 音频导演稿

## 目标

把已经合格的正文初稿，改成适合 TTS 生成的音频导演稿：更口语、更有停顿、更有轻情绪，但不改变新闻事实和产品宪法。

## 输入

- `稿子/日期_话题_小学档.md`
- 本目录的 `情绪playbook_v1.md`
- `手工跑通/03_音频生成与评估/MiniMax_播客音频生成研究与指导书_v0.1.md`

## 步骤

1. 先标出整期情绪曲线，而不是逐句贴标签。
2. 优先改台词语义、句式、标点和停顿。
3. 只在必要处加入 MiniMax 可解析提示，如 `（平静）`、`（快）`、`（压低）`、`<#0.45#>`。
4. 爸爸保持温和、递球、探索；孩子承担更多惊讶、追问和发现。
5. 选出最该短测的 6-8 个 turn。

## 产物

- `稿子/日期_话题_minimax导演稿_v1.md`

## 通过标准

- 去掉控制词以后，台词本身仍像真人对话。
- 情绪峰值有起有落，没有全篇兴奋或全篇低沉。
- 不出现 Fish 方括号标签。
- 不新增教育总结，不改变新闻事实。

## 沉淀位置

稳定的导演规则写回 `prompts/` 或本目录 playbook；只有确认为产品机制的内容才写回 `Writing_Kit.md`。
