# 迭代记录 v1

## 据review改了什么、为什么
- fableSection 由六把锁加到【七把锁】，新增最关键的「锁7·结构总结归孩子说」：三篇 review 一致命中同一处真违宪——爸爸用陈述句替孩子把寓言/模型的『结构骨架』点破了（『那口凹锅把后头全给定死了』『真正把过路车挤走的是那条拐弯规矩』『你听听你自己说的…这跟你一开始想的一样吗』）。原六把锁只封『道理总结』（应该怎样），没封『结构总结』（它是什么形状）；这一拍恰恰是整个形态的命门，必须钉死：结构句只能从孩子嘴里出，爸爸只许用反问把方向盘递过去。这是 review 里唯一够得上『削弱核心机制3』的系统性漏洞。
- menuSection 形态13『反推台』补『翻牌不许一口气倾倒』硬约束：review 指出反推台的翻牌段结构性地逼出『爸爸连讲3个事实』的信息倾倒，使全稿后1/3『问句多于结论』实质失衡（宪法3）。把『翻牌时至少留一个事实改成反问抛回孩子、让最反转那一记从孩子嘴里出』写进形态自身节拍，从源头堵住这个形态自带的违宪风险。同理给形态11『长镜头』翻牌、形态13都补了这一句。
- rotationRules 新增『收尾沉默族·硬去重』：三篇稿连续都用『爸爸放下/端起餐具+沉默不夸+孩子悬而未答』收尾——原收尾形状轮换表只按『具体形状』去重，没按『族』去重，导致换了形状却没换族（端碗→搁筷→摆勺都属同族）。新增：把⑦定格不接话/⑧停在孩子反问爸爸沉默/⑨⑩⑪悬空零件 全部归入『沉默留白族』，该族连续2期不得复用；逼下一期必须跳到『留观察小任务』或『代入式提问』等非沉默族。
- rotationRules 新增『落点形状轮换』：review 发现三期落点都是『A 还是 B』二元对峙式开放问题，正在变成可被孩子摸清的说理套路。新增按『落点形状』分族（①二元对峙②代入式『换你会…』③单点留白④去查/去看的开放任务⑤悬空画面定格），同形连续2期不得复用——开放问题的『骨架』也要错开，不只是换措辞。
- rotationRules 新增『揭盅—坐实接缝去重』：review 指出『先用虚构/模型载体玩透 → 再坐实「这真发生过+甩数字」』这个揭盅接缝在多期同味（潘多坐实名字、巴塞罗那坐实真城、寓言坐实数字）。新增：坐实点的位置与手法连续2期不得同款（要么把『这是真的』前置到开头、要么把惊奇从『竟然真有』挪到别处），防接缝本身变签名。
- rotationRules 十期自检表加一问『本期那句结构句/反转句，是爸爸说的还是孩子说的？』：把 review 反复命中的『爸爸划重点』变成一道可查的自检闸——只要全期最反直觉/最点题的那句结构句是从爸爸嘴里陈述出来的，判不合格、改为反问下放给孩子。这是把三篇 review 的共同教训固化成可执行检查，而不是停在原则。
- menuSection 角度17由『模糊备用位』改为明确的『角度族冲突时的降级取法』，与 rotationRules 的角度族去重表对齐，去掉落地时『可省』的暧昧，避免实操时无所适从。

## 结论
三篇验证稿 review 结论汇总：旧锅稿(寓言/路径依赖)=keep、反推台稿(深海鱼)=conditional keep、巴塞罗那造物台稿=revise。三篇 verdict 不同，但**指向同一处系统性病根**：稿子本身扎实、骨架够新、宪法 1/2/4/5/6/7 全守、新闻事实与 URL 均经 review 核验属实——问题集中在两条机制层缝隙上。(一) 三篇都在"最点题那一拍"让爸爸用陈述句替孩子把结构骨架说破了（"全给定死了""真正把车赶走的是那条规矩""你听听你自己说的"），削弱核心机制3，属灰区偏违宪。(二) 三篇连续用"爸爸沉默+小动作+孩子悬而未答"收尾、且落点都是二元对峙，签名手势正在成形。

这套 v0.4 增补针对性堵住了这两条缝：fableSection 加第七把锁「结构总结归孩子说」并写进自检命门第一问，把 review 最一致的违宪点钉死成可查闸门；rotationRules 加「沉默留白族整族去重」「落点形状轮换」「揭盅接缝去重」三层签名闸门；menuSection 给反推台/长镜头/造物台/分账桌的翻牌-揭盅拍都补了"不许倾倒、留一记反问给孩子"，从形态自身堵住信息倾倒型失衡。其余 review 建议（说理引擎近邻族去重、错别字、趣味疲点）也已分别落进引擎扩表与形态说明。

**结论：可上 prompt v0.4。** 这套增补不新增主线、不动已定设定、不给家长加新开关，逐条守宪法（新形态全程爸爸递材料/报麻烦/加规矩、台词问句驱动；落点停开放问号；概念名与北极星词在七把锁里零出现；十期滑窗+族去重直接服务"每期错开"）。落地动作：B-1/B-2/B-4 追加不替换，B-3 升级替换（旧赖账式降为单线支、四锁继承加严到七锁），迭代记录记「v0.4 — 形式与角度扩容：+5形态/+7角度/概念寓言一般化加第七把锁/三层签名闸门」，正文版本号改 v0.4。唯一需在 v0.5 继续观察的：第七把锁"结构总结下放给孩子"会否让部分孩子在反直觉那拍卡壳推不出——这要靠真实出稿+孩子反应回流校准，不在文档层解决。
