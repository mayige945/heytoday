## 二、逐篇 verdict 与关键问题

> 每篇过了两道镜：**宪法镜**（六铁律 + v0.2）与**形式与真实性镜**（形态是否真用出来、出处是否可核验）。

### 探索1 · fable（寓言）—— verdict：**keep**（两镜均 keep）

- **形态是否真用出来**：是，且差异化最彻底——故事先行、新闻后半程才浮出，大半篇幅活在虚构的"会自己回满的井"里。是 5 篇里唯一敢让虚构意象占主体的。
- **比附牵不牵强（诚实核验，这是寓言最易翻车处）**：**不牵强，同构精度高**。井（泉眼每晚只渗一桶 = 再生速率有限 / 各家只多舀一点点都说得通 = 个体理性 / 合起来超过再生就见底 = 集体非理性）与海洋鱼群严格同构，不是"鱼少了像井干了"的松散类比；连"越少越抢、越抢越少"的二阶挤兑动态都在井与海两边都成立——这是本质映射、不是表层比附。四把锁守住了：停在"井底见泥"画面未拖道理、抽掉道德标签（多舀的人/捕鱼的人都不是坏人）、未点"公地悲剧/系统思维"之名、回新闻是问号窄桥。
- **唯一可议**：L94"已经不只是在花利息、是在动本金"是全篇离"道理总结"最近的一句。但它是**新闻原文专家原话**（FAO 副总干事受访比喻）、用钱的画面而非概念词，不算爸爸以权威收口，**不构成违规**。打磨方向：可让孩子来接出"本金"二字，把结论权再还一步。
- **出处**：verified。已实测 https://news.un.org/zh/story/2025/06/1139441 ——35.5% 过度捕捞、77.2% 可持续、2570 个鱼群、尼斯第三届联合国海洋大会、"利息/本金"比喻均与原文相符，稿中"井"明确标注为虚构意象，事实与隐喻分层清楚。
- **结论**：寓言子形式**值得纳入菜单**，但红线必须钉死（已写进 prompt B-3 与菜单四把锁）。

### 探索2 · mystery（破案桌）—— verdict：**revise**（宪法镜 revise / 形式镜 keep）

- **形态是否真用出来**：是，**实战最强差异化**。开场不抛新闻而抛悬案（"先别吃，帮我破个案"），爸爸"只发线索、从不揭晓"，孩子三次翻案（划掉天气→锁海獭→因深水线索放下海獭→看画面才喊狼），揭晓靠"把手机递过去自己看"而非爸爸说出口——"答案权威"让渡得最干净。证据意识藏在"新线索来了，跟你刚才那猜还对得上吗"这个动作里，没点名。
- **为什么 revise（宪法镜点出的两处轻度越界）**：
  1. **L84"'我不确定'——这四个字，破案的时候比'我确定'值钱。"** ——全篇唯一以爸爸权威口吻说出的"X 比 Y 值钱"格言式价值判断，虽贴着破案动作、未点北极星词，但严苛看是一句微说教。**修正方向**：改成不带格言味的就事论事——爸爸只回"那就先别定"或反问"那你打算怎么办"，让"不确定先挂着"成为孩子的下一步动作，而不是爸爸盖章的道理。
  2. **L80"你这句话，比刚才那句'十有八九是海獭'稳多了"** ——爸爸对孩子推理方式作了好坏评级。**修正方向**：降一档为指向具体差别的反问（"这句和'十有八九'差在哪？"），把"谁的思路更好"也交还给孩子。
- **出处**：verified。Smithsonian + PMC 镜像两处交叉核到（加拿大 BC Heiltsuk 领地、清除外来欧洲绿蟹、鲱鱼饵、2024-05-29 母狼叼浮球顺绳拖笼、"狼如何学会"多种未定猜测）。
- **结论**：破案桌**值得纳入菜单**；这两句已并入 prompt 待改池 C 的处理，并在 B-1 形态 5 注记里写明"评级句改就事论事反问"。

### 探索3 · firstperson（第一人称代入）—— verdict：**keep**（宪法镜 keep / 形式镜 keep）

- **形态是否真用出来**：是。父子整段"钻进"鸟的身体活一趟，每个事实节点改写成第一人称**处境**而非旁白知识（"底下突然没了岸"而非"它飞越大海"）；爸爸自己也钻进去当"飞旁边那只 B6"。同理心/勇气只在"我替它害怕了一下"自然发生，全稿无一句道理。守住红线：拟人只到处境与感受，鸟没替人类喊一句口号。
- **可议**：骨架句"你现在是它"出现约 5 次。评审判定这是该形态**必需的代入锚点、不是该清的口头禅**，但 L43/51 密集处可换一两处"换你呢/你这会儿"换口气。"落水即死"是默认安全层、非红线，情绪有出口（最后活着落地）。
- **出处**：形式镜 verified（USGS/Guinness/Alaska Public Media 三链实测）。一处**出处归属偏差**（不影响事实真实，仅精度）：L91 把"飞偏后急转弯"挂在 Alaska Public Media 名下，但该细节实由 Guinness 页（sharp 90-degree turn）支撑——建议改挂或在 Guinness 标签补注。
- **结论**：第一人称代入**值得纳入菜单**，"骨架句不算口头禅"已写进 B-1 形态 7 注记。

### 探索4 · perspectives（多视角并置）—— verdict：**keep**（宪法镜 keep，附 2 处软违规待修 / 形式镜 keep）

- **形态是否真用出来**：是。同一辆没动的车被三方依次"看见"成三件不同的事，且**层层加码**——前两个是对立（嫌没用 vs 嫌用错地方），第三个把尺度从"这一辆"拉到"一整片"，是质变而非重复（多视角形态最难写好的一跳，做到了）。爸爸反复"我也判断不了谁对"，从不裁判。
- **两处软违规（宪法镜，待修但未推翻 keep）**：
  1. **L56** 爸爸以权威口吻独自给出第三视角的定论（"我后来才琢磨明白……对他来说……是第几辆"），未经孩子说出——**修正**：拆成追问，让孩子说出第三视角，与前两位一致。
  2. **L100** "你先别急着替谁说话。你就站那儿瞅一会儿" ——行为指令，句式贴近"你应该"红线——**修正**：软化为非指令。
- **形态风险**：中段广州"单车围城"/电子围栏/潮汐调度让"三目击者"短暂滑向**治理科普**，且把分属武汉/广州的做法泛化成"有的城市"。已用孩子的话拽回，但建议压缩（围栏与潮汐二选一），并在出处括注里点明归属。
- **出处**：verified。中国交通报《便民有序 精准高效——四地共享单车治理实践观察》（2025-09-17，zgjtb.com/2025-09/17/content_484720.html）实测存在，"单车围城"、广州投放超百万辆、电子围栏、武汉潮汐车组均可核验。三位路人是"依据真实治理矛盾搭出的情境人物"，已诚实标注。
- **结论**：多视角并置**值得纳入菜单**；"三视角要层层加码、别滑向科普、落点别读成选一个对的"已写进 B-1 形态 6 注记。

## 三、反套路评审结论（最重要的一节）：samenessRisk = **high**

> 两位反套路评审独立给出同一判：**形态多样性 = 及格，句式与节拍多样性 = 不及格，综合判 high。**

### 3.1 核心诊断：形态多样掩盖了骨架单一

表层 4 件衣服（寓言/破案/第一人称/多视角）确实不同，单看任意一篇都不平庸。**但深层六层骨架全锁死**，一个孩子连听 3–4 期几乎必然摸出"又是早饭桌上，爸爸抛个怪事，逼我猜，最后说他也不知道、丢个问题给我"：

1. **场景**：5 篇全在早餐桌、全是爸爸+孩子二人、无第三说话人。
2. **节拍**：抛怪物件/现象 → 孩子猜 → 爸爸不裁判反问 → 中段勾出新闻 → 爸爸"我也不知道"交权 → 落 1 个开放问题不收口。
3. **"我也不知道"位置**：5 篇全部钉在结尾问题正前方那一格（探索1 L110 / 探索2 L116 / 探索3 L71 / 探索4 L95 / 基线 L81），位置完全可预测。
4. **收尾动作**：3/5 篇是"孩子反抛问题 → 爸爸夸这个问题"（探索1"问到我心里去了"、探索4"这个问题比照片好"、探索2"你最想要下一条线索"）——"夸孩子的问题"已成新口头禅。
5. **饮品小节拍**：临近收尾都安插"喝/含一口"（粥/豆浆/奶）。
6. **说理引擎复用**：探索1（每家多舀一点点）与探索4（一个嫌少一个嫌多）用同一种"个体合理→系统失衡"推法。

### 3.2 最危险的发现：黑名单签名句已"借尸还魂"

基线被明令警惕的两句签名（"你猜怎么着""回来告诉我"）**没被当黑名单清掉，而是换皮渗进探索稿**——探索4 L24"结果你猜怎么——"、探索1 L30"你猜后来怎么着"、探索3 两处"你猜"；"回来告诉我"在探索1/3 以"回头咱俩对对"复活。**说明黑名单若不连变体一起封、且不强制轮换场景/开场/收尾/交权方式，套路化只会越收越紧。**

### 3.3 套路黑名单（已写进 prompt v0.3 的 B-4）

- **禁开场"你猜"族**：你猜怎么着 / 你猜后来怎么着 / 结果你猜怎么——
- **禁收尾"回来告诉我"族**：想到了回来告诉我 / 回头咱俩对对 / 回头告诉我 / 咱俩接着聊
- **禁爸爸被问住族**：把我问住了 / 问到我心里去了 / 把我问到头上了
- **禁捧场缴械族（单独拉黑，"夸孩子的问题"作为收尾动作整体禁止）**：这个问题问得好 / 你说得真准 / 你这话有意思 / 这个问题比……好
- **禁老签名句**：是不是很神奇 / 你想想看 / 这就是为什么 / 有意思吧
- **禁换皮说教**：我们学到 → 咱俩今天搞明白了（照删）

### 3.4 配套硬规则（已写进 B-4）

- **"我也不知道"**：每期 ≤1 次，禁钉在落点前一拍，轮换说法或这期不说改用递手机/让续写交权。
- **场景轮换池**：早餐桌/放学路上/睡前/超市排队/等公交/洗碗时/车里/散步，连续 2 期不同场景；允许第三人插话、允许孩子起话头；饮品小动作别每期都安。
- **开场配额**：≥6 种入口，每种连用不超 1 次，禁基线"今天爸爸看到一条新闻+你猜怎么着"。
- **收尾轮换**：8 种收尾形状，同一种连续 2 期不得复用；含"停在孩子反问上**爸爸沉默不夸**"（专治 3.1 第 4 条）。
- **说理引擎去重**：同一因果引擎 4 期内不得复用第 2 次。
- **节拍自检**：抽掉题材，本期节拍能否和上 2 期互换？可互换 → 判不合格、重排。

## 五、仍待用户拍板的开放问题

1. **待改池 C（离"教"最近的句子）要不要正式定调？** 破案桌 L84"我不确定比我确定值钱"、寓言 L94"利息/本金"是两个典型样本。当前 prompt 的处理是"评级句改就事论事反问、专家原话可保留但最好让孩子接出关键词"。是否要在 prompt 里明确写一条"允许/禁止"白名单，还是继续逐句观察？（评审意见：拍板权在主理人。）
2. **寓言子形式的适用边界够不够窄？** 现规定"机制类概念（个体小选择→集体非意图结果）才用"。是否担心写手会把它泛用到不该用的题材（具体真人/纯知识点/灾难新闻）？要不要加一道"每周至多 1 篇寓言"的配额？
3. **"情境化人物"的处理（探索4 三位路人、寓言的井）能否接受？** 两篇都用了"基于真实矛盾搭出的虚构情境人物/意象 + 真实可核验的冲突本身"，并诚实标注。这是否符合"严禁编造"的底线（评审判 verified、标注透明），还是要求当事人必须是新闻里的真人？
4. **场景轮换会不会冲淡"爸爸+孩子早餐对话"这个产品设定？** PRD/CLAUDE.md 把"早餐桌爸爸+孩子"近乎写成设定。B-4 要求轮换场景、允许第三人插话——这是反套路的必要手段，但与原设定有张力，需你确认放宽到什么程度（例如：核心仍是爸爸+孩子，场景可换、第三人只偶尔点缀）。
5. **B6 出处归属精度**：探索3 L91 的"急转弯"细节归属需从 Alaska Public Media 改挂 Guinness。这是探索稿层面的小修，是否要回填？（不影响 prompt。）

## 六、一句话总结

4 个形态全部 verdict 可用（fable/firstperson/perspectives = keep，mystery = revise 仅两句微说教待改），**寓言比附经核验不牵强、未滑向说教**。真正的病根不在形态而在**深层骨架与签名句**——反套路评审判 high。v0.3 的重心因此从"多给形态"转向"强制轮换句式/节拍 + 黑名单连变体封杀"，并把"抽掉题材节拍能否互换"设为新的不合格判据。

## 七、用户拍板结果（2026-06-30）

第五节 5 个开放问题，用户逐条拍板，已回改 prompt（见 `prompts/小学高年级_写稿提示词_v0.3.md` 迭代记录"v0.3 修订"条）：
