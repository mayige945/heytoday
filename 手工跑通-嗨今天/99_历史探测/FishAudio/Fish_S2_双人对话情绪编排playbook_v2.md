## 一句话策略

双人播客的情绪不是逐句贴标签，而是一条有起伏的探索线：

```text
轻松碰头 → 反常识钩子 → 困惑卡住 → 孩子试探猜 → 猜中后兴奋 → 事实推进 → 危机下沉 → 解决带来希望 → 更深问题反转 → 留白收束
```

Fish S2 标签负责“怎么演”，台词负责“为什么有这个情绪”，对话结构负责“什么时候起、什么时候落”。

## 核心原则

1. **情绪先来自对话动机，再写标签。**  
   先判断这句在做什么：抛问题、试探、被数字砸到、猜中、担心、反转、留白。再写 `[表演指令]`。

2. **标签写成导演提示，不写成抽象心情。**  
   少用 `[curious]`、`[sad]` 这种单词；多用 `[试探着猜，尾音上扬]`、`[担心，声音放轻]`。

3. **爸爸稳，孩子活。**  
   爸爸的情绪是温和、困惑、递球、压低、放慢；孩子的情绪是惊讶、追问、猜测、兴奋、担心。父子对比越清楚，播客越像真聊天。

4. **每一段只允许一个主情绪峰值。**  
   一直兴奋就会平，一直担心也会钝。需要中性句作为对比。

5. **口语化不是塞语气词。**  
   “啊、嗯、诶、咦、哈”只点睛。真正的口语感来自自我打断、复读、半句、追问、犹豫、改口。

6. **标签不能替代台词。**  
   去掉标签后，台词本身仍要像真人对话；标签只是把该有的读法放大。

## Fish S2 推荐语法

```text
<|speaker:0|>[温和，像自己也没完全想明白] 我昨天看到一条新闻，第一眼就卡住了。
<|speaker:1|>[一下来精神，带着笑意] 啊？还有你看不明白的？
```

使用边界：

- 情绪/表演方向：优先用 `[自然语言描述]`。
- 主情绪优先中文描述，并且标签后留一个 ASCII 空格。
- 英文短标签只用于局部声学效果或句中变化，如 `[whispering]`、`[gasping]`、`[sighing]`。
- 说话人切换：固定用 `<|speaker:0|>`、`<|speaker:1|>`。
- 生成路径：如果使用官方稳定多说话人模型，可整段一次生成；如果使用 `s2.1-pro-free`，不要依赖 `reference_id` 数组，改用逐句单 voice id 拼接。
- 口语停顿：优先写 `嗯`、`啊`、`……`、`——`。
- 实验型停顿/拟声：少量试 `(break)`、`(breath)`、`(sigh)`、`[gasp]`、`[laughing]`。
- 不再沿用 S1 的 `(happy)`、`(sad)` 作为主情绪写法。
- 避免 `[surprised][excited]` 这类多标签堆叠；需要复合情绪时写成一个中文标签：`[惊讶，又有点兴奋，声音上扬]`。

## 双人角色情绪职责

### 爸爸：稳态探索者

爸爸不是老师，也不是主持人腔。他是“自己也卡住了，所以拉孩子一起想”的人。

爸爸常用表演方向：

```text
[温和，像自己也在想]
[轻松开场，像刚坐下来聊天]
[放慢一点，把问题递出去]
[压低声音，像讲一个反常识发现]
[有点没转过来，但不急]
[担心，声音往下落]
[收住情绪，留一个问题]
```

爸爸避免：

```text
[权威讲解]
[兴奋演讲]
[教育孩子]
[非常激动地总结]
```

爸爸典型台词形态：

```text
我第一次看到这句，也没转过来。
你先别急着反驳，咱俩把它拆开看。
我卡住的是这儿：它凭什么算“一个”？
你这个说法，把我前面没通的地方，松开了一点。
```

### 孩子：情绪发动机

孩子承担更多听感起伏，但不是一直吵闹。他的变化应该来自“发现过程”。

孩子常用表演方向：

```text
[一下来精神，声音抬起来]
[惊讶，短促反问]
[试探着猜，像还没完全确定]
[突然想到了，语速快一点]
[恍然大悟，后半句变轻快]
[担心，声音放轻]
[慢慢想明白，声音放平]
```

孩子避免：

```text
[持续兴奋]
[一直大喊]
[假装可爱]
[每句都惊讶]
```

孩子典型台词形态：

```text
啊？一片树林，是一棵树？
等一下……除非，它们底下是连着的？
根呗！就像手指头分开，可底下连着一个手掌！
……那如果小树干一直长不起来，后面不就没人接上了吗？
```

## 细分情绪表达说明

### 1. 轻松碰头

用途：开场建立“今天又一起聊”的亲近感，不要像节目主持词。

表演方向：

```text
[轻松开场，像刚坐下来聊天]
[温和，带一点笑意]
```

台词手法：

- 用“来”“咱俩”“我昨天看到”进入。
- 不要“欢迎收听”“今天我们要学习”。

示例：

```text
<|speaker:0|>[轻松开场，像刚坐下来聊天] 来，咱俩今天聊个有点怪的新闻。
<|speaker:1|>[随口接话，带点好奇] 又是那种听起来不太像真的？
```

### 2. 反常识钩子

用途：第一分钟必须让孩子“啊？”一下。

表演方向：

```text
[压低声音，像讲一个反常识发现]
[不急着揭晓，前半句放慢]
```

台词手法：

- 先给具体事实，再给反常识判断。
- 数字、对比词要清楚。

示例：

```text
<|speaker:0|>[压低声音，像讲一个反常识发现] 美国有一片树林，四万七千多根树干，科学家却说，它其实是一棵树。
<|speaker:1|>[惊讶，短促反问] 啊？一片树林，是一棵树？
```

### 3. 困惑卡住

用途：让爸爸暴露自己的不懂，把孩子请进来。

表演方向：

```text
[有点没转过来，但不急]
[放慢一点，把困惑说清楚]
```

台词手法：

- 用“我卡住的是这儿”。
- 把不懂具体化成一个小问题。

示例：

```text
<|speaker:0|>[有点没转过来，但不急] 我卡住的是这儿：四万七千根树干，凭什么能算“一个”？
```

### 4. 试探猜测

用途：让孩子不是被动听答案，而是在现场推理。

表演方向：

```text
[试探着猜，像还没完全确定]
[尾音微微上扬]
```

台词手法：

- 用“除非……”“是不是……”“会不会……”。
- 句尾用问号，不要立刻说成结论。

示例：

```text
<|speaker:1|>[试探着猜，尾音上扬] 除非……它们底下是连着的？
<|speaker:0|>[顺着追问，放慢] 你说底下，什么连着？
```

### 5. 猜中兴奋

用途：孩子自己发现关键结构，形成第一个情绪峰值。

表演方向：

```text
[突然想到了，语速快一点]
[兴奋，但别喊]
```

台词手法：

- 用“根呗！”这类短促确认。
- 用生活比喻，句子可以稍微连起来，形成“一口气说出”的感觉。

示例：

```text
<|speaker:1|>[突然想到了，语速快一点] 根呗！就像手指头是分开的，可底下连着一个手掌。那是一只手，不是五只手！
```

### 6. 恍然松开

用途：第一个谜题解开后，情绪从峰值落回稳定。

表演方向：

```text
[慢慢想明白，声音放平]
[温和，像被孩子点通了]
```

台词手法：

- 爸爸不要夸“你真聪明”，用“我那没通的劲儿松了点”。
- 孩子可以用“哦……”承接。

示例：

```text
<|speaker:0|>[温和，像被孩子点通了] 你这么一捋，我那没读通的劲儿，松了点。
```

### 7. 事实推进

用途：进入第二层事实，不要让情绪一直停在兴奋。

表演方向：

```text
[平稳推进，像继续翻下一页]
[放慢一点，准备引出新问题]
```

台词手法：

- 爸爸继续念事实，孩子继续判断。
- 多用短句，减少标签。

示例：

```text
<|speaker:0|>[平稳推进，像继续翻下一页] 可它后面又说，这棵树已经冒了几千年，有人估到一万多年。
<|speaker:1|>[惊讶，声音上扬] 一万年？一棵树能活一万年？
```

### 8. 危机下沉

用途：从“真奇妙”进入“出事了”，制造第二个低谷。

表演方向：

```text
[担心，声音往下落]
[慢下来，像看到后果]
[声音放轻，不煽情]
```

台词手法：

- 通过画面表达担心：老树干、小树干找不着、后面没人接。
- 不讲大道理，不上升到环保口号。

示例：

```text
<|speaker:0|>[担心，声音往下落] 这条新闻说，潘多现在出事了。老树干一大片，可刚冒头的小树干，几乎找不着。
<|speaker:1|>[声音放轻，慢一点] ……那等这些老树干一根一根倒下去，后面不就没人接上了吗？
```

### 9. 发现原因

用途：让孩子继续推理，而不是爸爸直接揭示。

表演方向：

```text
[试探着猜，带一点紧张]
[突然想到原因，语速稍快]
```

台词手法：

- 从“根没死”推到“小芽被吃掉”。
- 让孩子提出“是不是被谁吃了”。

示例：

```text
<|speaker:1|>[试探着猜，带一点紧张] 它根又没死……是不是新芽冒出来，被谁吃掉了？
```

### 10. 希望回升

用途：围栏有效，情绪短暂回升，但不要变成大团圆。

表演方向：

```text
[轻快一点，像看到转机]
[带一点希望，但别太满]
```

台词手法：

- 用“真长起来了”制造小亮点。
- 孩子问“那它有救了？”保持不确定。

示例：

```text
<|speaker:0|>[轻快一点，像看到转机] 真长起来了。围栏里头，没几年，新树干蹿得老高。
<|speaker:1|>[带一点希望，但别太满] 那它有救了？
```

### 11. 更深问题反转

用途：从事实问题转向思考问题，形成最后的深度。

表演方向：

```text
[收住轻快，慢慢转向更深的问题]
[轻一点，像突然意识到问题不简单]
```

台词手法：

- 用“可是我又想起你前头那句”转弯。
- 不要说“所以我们得出一个哲学问题”，要让问题自然浮出来。

示例：

```text
<|speaker:0|>[收住轻快，慢慢转向更深的问题] 可我又想起你前头那句：它四万七千根，是一个，一套根，一个脾性。
<|speaker:1|>[慢慢想明白，声音放低] 那要是来一种病，正好是它扛不住的那种……
```

### 12. 留白收束

用途：不下标准答案，把问题留给孩子。

表演方向：

```text
[收住情绪，留一个问题]
[慢慢说，尾音落下去]
```

台词手法：

- 爸爸提出最后问题。
- 孩子给一个“两头看”的开放回答。
- 爸爸只轻收，不总结教育意义。

示例：

```text
<|speaker:0|>[收住情绪，留一个问题] 它到底是一棵活了一万年的老树，还是一万根各自只活一阵、轮流站岗的小树？
<|speaker:1|>[慢慢说，像自己也还在想] 那得看你打哪头看吧。从地底下那张根看，是一个；从一根一根的树干看，又是好多个。
<|speaker:0|>[轻轻收住] 嗯。明天见。
```

## 整期情绪叙事曲线

### 标准 10-15 分钟结构

| 阶段 | 情绪能量 | 主导者 | 作用 | 标签方向 |
|---|---:|---|---|---|
| 1. 碰头 | 2/5 | 爸爸 | 建立亲近 | `[轻松开场]` |
| 2. 反常识钩子 | 4/5 | 爸爸→孩子 | 抓住注意力 | `[压低声音]`、`[惊讶]` |
| 3. 卡住 | 3/5 | 爸爸 | 暴露困惑 | `[有点没转过来]` |
| 4. 试探 | 3/5 | 孩子 | 进入推理 | `[试探着猜]` |
| 5. 猜中 | 5/5 | 孩子 | 第一个峰值 | `[突然想到了]` |
| 6. 松开 | 2/5 | 爸爸 | 情绪回落 | `[温和]` |
| 7. 新事实 | 3/5 | 爸爸 | 推进信息 | `[平稳推进]` |
| 8. 危机 | 1/5 | 爸爸→孩子 | 下沉 | `[担心]`、`[声音放轻]` |
| 9. 原因 | 3/5 | 孩子 | 再次推理 | `[带一点紧张]` |
| 10. 转机 | 4/5 | 爸爸→孩子 | 希望回升 | `[轻快一点]` |
| 11. 深层反转 | 2/5 | 爸爸 | 变深 | `[慢慢转向更深的问题]` |
| 12. 留白 | 1/5 | 父子 | 不收口 | `[收住情绪]` |

## 标签密度建议

一整期不要每句都加标签。推荐：

- 开场 1-2 个标签。
- 反常识钩子 2-3 个标签。
- 推理段每 3-5 句一个标签。
- 情绪峰值句必须有标签。
- 中性事实句尽量不加标签。
- 危机和留白用少量低能量标签。

经验比例：

```text
40% 无标签自然口语
35% 中文轻标签：温和、试探、放慢、带笑
20% 中文明显情绪标签：惊讶、兴奋、担心、希望
5% 英文短局部效果：gasping、laughing、whispering、shouting、sighing
```

## 强声学效果使用边界

可用但要少：

```text
[gasping]   # 真倒吸气，适合反常识或突发发现
[laughing]  # 真笑，适合孩子忍不住笑，不适合每个幽默点
[sighing]   # 叹气，适合危机或无奈
[whispering]# 讲秘密、压低反转
[shouting]  # 真喊，儿童播客极少用
```

示例：

```text
<|speaker:1|>我打开资料一看，[gasping] 它居然不是四万七千棵树。
<|speaker:0|>[whispering] 最怪的是，地底下那张根，可能才是真正的“它”。
```

## 反例与修正

### 反例 1：标签替代台词

差：

```text
<|speaker:1|>[非常惊讶] 怎么会这样？
```

改：

```text
<|speaker:1|>[惊讶，短促反问] 啊？四万七千根……怎么会是一棵？
```

### 反例 2：爸爸像老师

差：

```text
<|speaker:0|>[自信讲解] 这说明生物的个体概念非常复杂。
```

改：

```text
<|speaker:0|>[温和，像自己也在想] 我卡住的就是这儿：它到底算一个，还是算很多个？
```

### 反例 3：孩子一直兴奋

差：

```text
<|speaker:1|>[兴奋] 哇！
<|speaker:1|>[兴奋] 太厉害了！
<|speaker:1|>[兴奋] 然后呢！
```

改：

```text
<|speaker:1|>[惊讶，短促反问] 啊？一片树林，是一棵树？
<|speaker:1|>[试探着猜，尾音上扬] 除非……它们底下是连着的？
<|speaker:1|>[突然想到了，语速快一点] 根呗！像手指头那样！
```

### 反例 4：结尾教育化

差：

```text
<|speaker:0|>[认真总结] 所以我们要保护自然，也要学会从不同角度看问题。
```

改：

```text
<|speaker:0|>[收住情绪，留一个问题] 那你说，它到底是一棵活了一万年的老树，还是很多根轮流站岗的小树？
```

### 反例 5：标签格式不稳

差：

```text
<|speaker:1|>[surprised][excited]啊？盒子里有星星？
<|speaker:1|>[一下子兴奋，声音上扬]啊？盒子里有星星？
```

改：

```text
<|speaker:1|>[一下子兴奋，声音上扬] 啊？盒子里有星星？
<|speaker:1|>我打开盒子，[gasping] 里面真的有一颗小星星！
```

## 写稿时的操作流程

1. 先写无标签对话，确认去掉标签也像真人聊天。
2. 给每一段标一个主情绪：轻松、困惑、试探、兴奋、担心、希望、留白。
3. 只给关键句加 Fish S2 标签。
4. 检查爸爸是否稳，孩子是否活。
5. 检查整期曲线是否有上有下，不能全程一个能量。
6. 生成短样本先听标签是否有效，再生成整期。
7. 如果听感平，优先改台词和标签具体度，再考虑换参考音。

<|speaker:0|>[轻松开场，像刚坐下来聊天] 来，咱俩今天聊个有点怪的新闻。

<|speaker:1|>[惊讶，短促反问] 啊？一片树林，是一棵树？
<|speaker:0|>[有点没转过来，但不急] 我卡住的也是这儿。四万七千根树干，凭什么能算“一个”？
<|speaker:1|>[试探着猜，尾音上扬] 除非……它们底下是连着的？
<|speaker:0|>[顺着追问，放慢] 你说底下，什么连着？
<|speaker:1|>[突然想到了，语速快一点] 根呗！就像手指头是分开的，可底下连着一个手掌！
<|speaker:0|>[温和，像被孩子点通了] 你这么一捋，我那没读通的劲儿，松了点。

<|speaker:1|>[带一点希望，但别太满] 那它有救了？
<|speaker:0|>[收住轻快，慢慢转向更深的问题] 可我又想起你前头那句：它们是一个，一套根，一个脾性。
<|speaker:1|>[慢慢想明白，声音放低] 那要是来一种病，正好是它扛不住的那种……
<|speaker:0|>[收住情绪，留一个问题] 所以我最后卡在这儿：它到底是一棵活了一万年的老树，还是很多根轮流站岗的小树？

## 与旧版 playbook 的关系

`../../02_音频导演稿/情绪playbook_v1.md` 的价值仍在：它强调“情绪必须写进台词语义、句式和标点”，这个判断继续成立。

本版修正的是：旧版把 Fish 标签能力判断得过弱，只保留少数强声学标签。最新结论是，Fish S2 / S2.1 的自然语言方括号指令足够有用，应当成为脚本情绪化编排的一等工具。但它必须和台词、标点、角色职责、整段情绪曲线一起使用，不能孤立堆标签。
