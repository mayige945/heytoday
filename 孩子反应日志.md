# 孩子反应日志

> **这是什么**：把"出稿 → 看孩子反应 → 改 prompt → 次日再看"这个循环里**最关键、又最容易丢**的一环——孩子的真实反应——记下来。N=1 手动跑，唯一的真信号来自这里。
> **怎么用**：每放完一期，趁热（当天 / 最迟次日早餐后）填一格。**别等攒一周再补**，记忆会骗你。填的时候只记**观察到的事实**，别急着下结论（"他走神了"是事实，"这段太枯燥"是你的解释，写进「我的解读」那栏）。
> **它喂给谁**：这份日志 → 校准 `prompts/小学高年级_写稿提示词` 的下一个小版本（方法论第六步）。每条改 prompt 的决定，最好能在这里指到一格具体反应。

## 怎么填（四个核心信号 + 一句解读）

每期至少抓这四个，PRD/方法论里反复强调的就是它们：

1. **完播**：听完了没？中途喊停 / 走开了没？（最硬的信号——没听完，别的都白搭）
2. **走神点**：哪一句 / 哪一段眼神飘了、开始抠别的、插不相干的话？（标到段落）
3. **主动追问**：他自己冒出来的问题 / 接话 / "那……"。**这是金信号**——说明稿子真把他的脑子勾动了，不是被动听完。原话尽量抄下来。
4. **次日意愿**：第二天他还想不想听？主动提没提昨天那期？（决定这档能不能"累积"下去）

加一栏 **我的解读 + 给 prompt 的动作**：把上面的事实翻成"下一版 prompt 该不该动、动哪条"。能靠调一条原则解决的，别堆细则（方法论原则 2）。
